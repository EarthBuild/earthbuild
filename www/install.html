Install EarthBuild

Get EarthBuild running on your machine in under a minute.

Prerequisites

EarthBuild requires Docker to be installed and running:

  • macOS: Docker Desktop for Mac
  • Linux: Docker Engine
  • Windows: Docker Desktop for Windows with WSL 2

macOS

Install with Homebrew:

brew install earthbuild/tap/earthbuild

Or download directly:

curl -fsSL https://get.earthbuild.dev | sh

Linux

Install with our script:

curl -fsSL https://get.earthbuild.dev | sh

Or install manually:

# Download the latest release
curl -LO https://github.com/EarthBuild/earthbuild/releases/latest/download/earthbuild-linux-amd64

# Make it executable
chmod +x earthbuild-linux-amd64

# Move to your PATH
sudo mv earthbuild-linux-amd64 /usr/local/bin/earthbuild

Windows (WSL 2)

Inside your WSL 2 environment:

curl -fsSL https://get.earthbuild.dev | sh
Note: Native Windows support is experimental. We recommend using WSL 2.

Verify Installation

earthbuild --version

Bootstrap EarthBuild (first-time setup):

earthbuild bootstrap
